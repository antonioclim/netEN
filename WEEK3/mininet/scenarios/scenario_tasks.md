# Mininet scenario tasks — Week 3

These tasks are designed for the Week 3 topology.

## Topology
Client ↔ router ↔ server

## Tasks
1. Start the topology, confirm addressing and routing
2. Run the TCP tunnel demo and confirm forwarding works end-to-end
3. Run a UDP broadcast example and explain where it is received
4. Run a UDP multicast example and confirm group membership behaviour (IGMP)
5. Capture traffic and extract at least:
   - one UDP broadcast datagram
   - one IGMP membership message
   - one TCP handshake packet
