# Assessment rubric (Week 1)

Your submission is assessed using the following criteria.

## Reproducibility
- runs on a clean Linux CLI-only VM
- produces the expected artefacts with the provided scripts
- passes the smoke test without manual intervention

## Technical correctness
- commands and outputs match the described steps
- ports and paths follow the kit conventions
- captures and logs are consistent with the demo

## Clarity
- `README.md` is followed exactly
- artefacts are easy to locate and interpret
