# Rubrici of Evaluation — Week 13

## Laboratory Criteria (15p of final grade)

### Scanning Ports (3p)
- 3p: All ports identified correctly, formatted output
- 2p: Ports identified, minimal output
- 1p: Partially functional
- 0p: Does not work

### MQTT Client (3p)
- 3p: Pub/sub functional, calculated statistics
- 2p: Basic communication functional
- 1p: Connection successsful, partial communication
- 0p: Does not connect

### Capture Traffic (3p)
- 3p: Complete capture, correct analysis
- 2p: Capture saved, superficial analysis
- 1p: Capture only, without analysis
- 0p: Missing capture

### Documentare (3p)
- 3p: Complete report with all sections
- 2p: Partial report
- 1p: Only answers to questions
- 0p: Undocumented

### Participare (3p)
- 3p: Activee, relevant questions
- 2p: Present, moderate participation
- 1p: Present, minimal participation
- 0p: Absent

## Team Project Criteria — Security Audit (10p)

- Exposed ports list (2p)
- Credential verification (2p)
- Communication encryption (2p)
- Logging configured (2p)
- Remediation plan (2p)
