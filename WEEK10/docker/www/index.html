Week 10 - Network Services Demo

This page is served by the Docker web service using Python http.server.

Quick checks

  • HTTP: curl -v http://localhost:8000/
  • DNS (custom server): dig @127.0.0.1 -p 5353 myservice.lab.local
  • SSH: ssh -p 2222 labuser@127.0.0.1 (password: labpass)
  • FTP: ftp 127.0.0.1 2121 (user: labftp, password: labftp)

Use make run-all to execute an automated end-to-end demonstration and generate artefacts.
