# Week 13 checklist

## Before you start

Ensure you are on a Linux VM with Python 3 and pip. Install requirements with `make setup`.

## Run the demo

Run `make demo` and confirm `artifacts/demo.log` and `artifacts/validation.txt` exist.

## Run tests

Run `make test` and ensure it exits with status 0.

## Cleanup

Run `make clean` and confirm Mininet is cleaned if it was used.
