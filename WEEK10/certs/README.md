# SSL/TLS Certificates

This directory stores SSL/TLS certificates generated for HTTPS demonstrations.

Certificates are generated automatically by `scripts/setup.sh` and cleaned by `scripts/cleanup.sh`.

## Contents (after setup)

- `server.crt` - Server certificate
- `server.key` - Server private key
- `test_server.crt` - Test certificate
- `test_server.key` - Test private key
