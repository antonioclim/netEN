# Week 6 Checklist

Use this checklist before you start and again before you submit.

## Environment

- [ ] Linux VM, CLI-only workflow
- [ ] `python3` and `pip3` installed
- [ ] Mininet installed and working: `sudo mn --test pingall`
- [ ] `tcpdump` installed (optional but recommended)
- [ ] Docker installed if you use the containerised demo

## Run

- [ ] `make setup` completes without errors
- [ ] `make demo` produces `artifacts/demo.log` and `artifacts/validation.txt`
- [ ] `make test` passes

## Submission

- [ ] `artifacts/demo.log`
- [ ] `artifacts/validation.txt`
- [ ] Short answers from `docs/lab.md` (commands and observations)
